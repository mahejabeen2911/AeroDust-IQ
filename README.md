# AeroDust-IQ

AeroDust-IQ is a smart project made to reduce dust pollution caused by heavy vehicles in the Raipur-Bhilai industrial area.

This system uses aerodynamic design, smart dust control, and real-time simulation to reduce dust coming from trucks carrying materials like coal, sand, and iron ore.

# Features

- Real-time vehicle simulation  
- Dust reduction system  
- AQI based monitoring  
- Pollution optimization dashboard  

# Technology Used

- HTML  
- CSS  
- JavaScript
  
# Project Goal

The main goal of this project is to control dust pollution from transport vehicles and help improve air quality in industrial areas.
